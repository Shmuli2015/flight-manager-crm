# Flight Manager

## Overview
Flight Manager is a web application designed to help manage flight operations, clients, and payments. It provides a comprehensive dashboard and tools for tracking flight-related information.

## Features

### Dashboard
- **Quick Stats Overview**
  - Total Clients
  - Upcoming Flights
  - Unpaid Payments
  - Completed Flights

### Navigation
- Responsive navigation with mobile and desktop views
- Easy access to key sections:
  - Dashboard
  - Clients
  - Flights

### Functionality
- Add New Clients
- Add New Flights
- Track Flight Status
- Monitor Payments

## Technology Stack
- React
- Vite
- TypeScript
- Tailwind CSS
- Lucide Icons
- Responsive Design

## Getting Started

### Prerequisites
- Node.js
- npm or yarn
- Vite

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Shmuli2015/flight-manager-crm.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
